# E-commerce - Sistema de recomendações

Uma aplicação web que exibe perfis de usuários e listas de produtos, com a capacidade de rastrear compras de usuários para futuras recomendações de aprendizado de máquina usando TensorFlow.js.

## Instalação e execução

1. Dependências:
```
npm ci
npm install
```

2. Rodar app:
```
npm start
```

3. Abrir `http://localhost:8080`

## Features

- Seleção de perfil de usuário com exibição de detalhes
- Histórico de últimas compras
- Lista de produtos para "comprar"
- Rastreamento de compras usando sessionStorage do navegador
